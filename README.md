## Overview

This virtual work experience simulates the day-to-day responsibilities of a Cyber Incident Response Analyst at STC, one of the largest telecom companies in the Middle East. The program is offered through the Misk Hub platform in partnership with STC's cybersecurity team.

Over the course of the experience, I investigated real-world style security incidents using industry-standard tools, produced formal incident reports, and developed a cybersecurity awareness campaign — all tasks that reflect actual SOC analyst workflows.
